KS (Korean Script) 문서

KS는 한국어로 배우는 학습용 프로그래밍 언어입니다.
프로그래밍이 처음인 사람도 쉽게 이해할 수 있도록 만들어졌습니다.

💡 KS는 인터프리터 언어이며, 현재는 웹에서 실행됩니다.

1. 출력하기

출력()은 화면에 글자를 보여줍니다.

출력("안녕")
출력(123)

2. 변수

변수는 값을 저장하는 상자입니다.

나이 = 20
출력(나이)
변수 이름은 한글, 영어, 숫자를 사용할 수 있습니다.
단, 숫자로 시작할 수는 없습니다.

3. 입력

입력()은 사용자에게 값을 입력받습니다.

이름 = 입력("이름을 입력하세요: ")
출력(이름)

4. 계산

KS는 기본적인 사칙연산을 지원합니다.

a = 10
b = 20
출력(a + b)
출력(a * b)

5. 조건문 (만약)

나이 = 15

만약 (나이 >= 20) {
  출력("성인입니다")
} 아니면 {
  출력("미성년자입니다")
}

6. 반복문

반복 (3) {
  출력("안녕하세요")
}

7. 에러 메시지

KS의 에러 메시지는 최대한 친절하게 보여줍니다.

❌ 3번째 줄에서 오류가 있어요
'이' 은(는) 사용할 수 없는 문자예요
에러가 나면 줄 번호와 이유를 함께 확인하세요.

8. 앞으로 추가될 기능

  • 콘솔 실행 (Python처럼)
  • IDLE 형태 실행기
  • 파일 저장 / 불러오기
  • 초보자 모드 / 전문가 모드
